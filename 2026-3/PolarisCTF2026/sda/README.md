# sda

## 核心识别

表面上每条关系都带平方项，但密钥本身只依赖平方值，所以自然代换成 `u_i = x_i^2` 和 `v = y^2`。代换之后，三条式子都会变成共享同一个小变量 `v` 的线性小误差系统。

另外，`A_i` 都能直接分解，因此 `phi(A_i)` 对攻击者是公开量。于是整个问题就能写成

`B_i * u_i - phi(A_i) * v = z_i`

并且误差 `z_i` 明显小于主项。这个结构非常适合低维嵌入格，根本不需要暴力，也不需要高阶 BKZ。

## 最短解链

1. 分解三个 `A_i`，求出 `phi(A_i)`。
2. 做变量代换，把题目改写成线性小误差系统。
3. 按误差上界构造 4 维格基，对 `(z1, z2, z3, v)` 做缩放嵌入。
4. 跑一次 LLL 拿到候选 `v`，再回代恢复三个 `u_i`。
5. 用 `v + u1*u2*u3` 派生 AES key，解出明文。
