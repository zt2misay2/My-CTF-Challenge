# ECC

## 核心识别

题目给的是广义 Weierstrass 形式，不是正常安全椭圆曲线。把方程配方后可化到奇异曲线，最终退化成可参数化的加法群。

关键代换是 `Y = 2y + c`，曲线会落到 `Y^2 = 4(x-r)^3`。对这类尖点曲线可以引入参数 `u = 2(x-r)/(2y+c)`，并满足 `u(P+Q) = u(P) + u(Q)`。于是标量乘直接线性化，最后只要算出 `m = u(P) / u(G) mod p` 即可。

## 最短解链

1. 识别三次多项式判别式为 `0`，说明曲线奇异。
2. 用常数 `r` 把曲线平移到尖点形式。
3. 把 `G` 和 `P = mG` 都映射到参数 `u`。
4. 直接计算 `m` 并转回字节串得到 flag。
