# smx

## 核心识别

只看导数值本身，常数 `c` 已经被求导消掉，题目其实不够解。真正让题目闭合的隐藏条件是：给出的 `x1` 不只是求导取值点，同时也是原三次多项式的一个根。

有了这个条件，路线就很直接了：

- 由 `deriv1_num = 3*x1^2 - 2*a1*x1 + b1`，结合 `a1`、`b1` 的取值范围，唯一恢复 `a1` 和 `b1`
- 由 `f(x1)=0` 得到 `c = x1^3 - a1*x1^2 + b1*x1 - k*n`
- 再枚举 8-bit 素数 `k`

因为题目实际用的是 `e = c^{-1} mod phi(n)`，所以 `cipher^c mod n` 就会直接回到明文。

## 最短解链

1. 用参数范围从导数值唯一确定 `a1` 和 `b1`。
2. 枚举所有可能的 8-bit 素数 `k`。
3. 对每个 `k` 计算候选 `c`。
4. 尝试 `m = cipher^c mod n`，检查是否长成 `xmctf{...}`。
